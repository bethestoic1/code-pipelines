- **Length:** 1,500–2,500 words for comparisons; 1,000+ for how-tos.
- **Structure:** Intro → comparison or steps → “Our take” / “Best for X” → CTA box.
- **CTA box (every article):**
  - “Compare more tools: [Link to /blog/]”
  - “We recommend BrainGrid for Cursor/Claude Code users. [Try BrainGrid →]” (your affiliate link)
  - Add other affiliate links only when relevant (e.g. Tabnine in Tabnine vs Cursor, Kinsta in hosting).
- **SEO:** Meta description, canonical, OG/Twitter, JSON-LD Article. Add new URLs to `sitemap.txt`.
